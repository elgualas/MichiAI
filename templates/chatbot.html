Tecnologías usadas a lo largo del desarrollo del proyecto

OpenIA

Se ha utilizado el modelo de lenguaje text-davinci-003 de OpenAI para implementar a "Michi-AI", un chatbot con capacidad de respuesta corta, sarcástica e irónica. Este modelo utiliza la técnica de aprendizaje profundo para generar respuestas en función del contexto y las preguntas proporcionadas.

Django

Se ha utilizado para construir la estructura y lógica de tu página web, incluyendo la implementación del chatbot, la cabecera, pie de página y otros componentes. Django ofrece un enfoque práctico y eficiente para el desarrollo web, proporcionando herramientas para la gestión de bases de datos, autenticación de usuarios, manejo de formularios y más.

Aseprite

Es una herramienta de edición y animación de gráficos pixelados especialmente diseñada para la creación de arte 2D, sprites y animaciones. Se ha utilizado para dibujar y animar el gato que actúa como el avatar de "Michi-AI" durante las respuestas del chatbot.
